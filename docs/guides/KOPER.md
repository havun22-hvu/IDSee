# Handleiding voor Kopers

## Waarom IDSee?

Bij de aankoop van een pup wil je zekerheid over:
- Is de fokker erkend en betrouwbaar?
- Komt de pup echt uit Nederland?
- Is de moeder geregistreerd?
- Zijn er gezondheidsgegevens beschikbaar?

IDSee geeft je deze zekerheid **zonder** dat je privégegevens van de fokker te zien krijgt.

---

## Pup Verifiëren

### Wat je nodig hebt
- Chipnummer van de pup (vraag aan fokker/dierenarts)
- Smartphone of computer

### Stappenplan

```
1. Ga naar idsee.app/verify
2. Voer chipnummer in: 528-XXXX-XXXX-XXXX
3. Klik "Verifiëren"
4. Bekijk resultaat
```

### Resultaat Uitleg

| Status | Betekenis |
|--------|-----------|
| ✅ Erkende Fokker | Fokker is gekeurd en geregistreerd |
| ✅ Moeder Bekend | Moeder is gechipt en geregistreerd |
| ✅ Professionele Registratie | Dierenarts heeft chip geplaatst |
| ⚠️ Niet Gevonden | Pup niet in systeem |
| ❌ Ongeldige Fokker | Fokker geschorst of onbekend |

---

## Gezondheidsgegevens Opvragen

### Hoe werkt het?

1. Vraag de fokker om toegang
2. Fokker deelt via IDSee app
3. Je ontvangt een link/QR code
4. Bekijk beschikbare records

### Wat kun je zien?

- **Type record** (vaccinatie, gezondheidscheck, etc.)
- **Datum** van het record
- **Dierenarts** (geverifieerd, naam niet zichtbaar)
- **Document** (indien gedeeld door fokker)

### Privacy

Je ziet ALLEEN wat de fokker met je deelt. Dit beschermt:
- Fokker privacy
- Medische details die niet relevant zijn
- Historische eigenaren

---

## Red Flags

Let op deze waarschuwingssignalen:

| Signaal | Risico |
|---------|--------|
| "Chip nog niet geregistreerd" | Mogelijk illegale import |
| Fokker wil geen verificatie | Verbergt iets |
| Moeder niet bekend | Broodfokker/puppymill |
| Geen gezondheidsrecords | Geen veterinaire zorg |

## Na Aankoop

### Eigendom Registreren (optioneel)

```
1. Download IDSee app
2. Maak Cardano wallet
3. Vraag fokker om overdracht
4. Accepteer in app
5. Pup staat nu op jouw naam
```

### Voordelen van Registratie

- Bewijs van eigendom
- Dierenarts kan records toevoegen
- Verzekering kan gezondheid verifiëren
- Bij verlies: bewijs dat het jouw dier is

---

## Voor Verzekeringen

IDSee kan ook door verzekeringen gebruikt worden:

```
1. Klant geeft toestemming
2. Verzekeraar vraagt verificatie
3. IDSee geeft cryptografisch bewijs:
   - "Dier heeft vaccinaties" ✅
   - "Geen bekende erfelijke aandoeningen" ✅
4. Geen medische details gedeeld
```

## Veelgestelde Vragen

**V: Kost verificatie geld?**
A: Nee, verificatie is gratis.

**V: Kan ik zien wie de fokker is?**
A: Nee, alleen DAT de fokker erkend is.

**V: Wat als de pup niet in het systeem staat?**
A: Vraag de fokker om registratie. Geen registratie = mogelijk risico.

**V: Kan ik een pup uit het buitenland verifiëren?**
A: IDSee werkt momenteel alleen voor NL-geregistreerde fokkers.

**V: Is dit 100% betrouwbaar?**
A: Het systeem voorkomt fraude op blockchain-niveau. De initiële registratie is afhankelijk van de dierenarts.
