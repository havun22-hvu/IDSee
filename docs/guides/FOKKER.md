# Handleiding voor Fokkers

## Wat is IDSee?

IDSee is een systeem waarmee je als erkende fokker de afkomst van je pups kunt bewijzen, zonder je persoonlijke gegevens te onthullen.

---

## Registratie als Fokker

### Vereisten

1. **Erkenning** door een van deze organisaties:
   - Raad van Beheer op Kynologisch Gebied
   - FCI erkend stamboek
   - KvK-registratie als fokker

2. **Cardano Wallet** (Nami, Eternl, of Lace)

3. **Inspectie** door een IDSee BreederInspector

### Stappenplan

```
1. Maak een Cardano wallet aan
2. Vraag keuring aan via IDSee app
3. Inspector bezoekt je en verifieert:
   - Fokkerij locatie
   - Erkenningsdocumenten
   - Moederdieren
4. Na goedkeuring: je bent geregistreerd!
```

---

## Pup Registreren

### Wanneer?
- Na chippen door dierenarts/erkende chipper
- Binnen 7 dagen na geboorte (aanbevolen)

### Benodigde gegevens
- Chipnummer pup (15 cijfers)
- Chipnummer moeder
- Geboortedatum
- Ras

### Proces

```
1. Dierenarts plaatst chip
2. Dierenarts opent IDSee app
3. Dierenarts scant chip
4. Dierenarts selecteert jouw fokkerij
5. Pup is geregistreerd op blockchain!
```

## Privacy

### Wat is NIET zichtbaar?
- Je naam
- Je adres
- Je KvK-nummer
- Exacte locatie fokkerij

### Wat is WEL verifieerbaar?
- "Deze pup komt van een erkende fokker" (JA/NEE)
- "De moeder is geregistreerd" (JA/NEE)
- "Geregistreerd door erkende dierenarts" (JA/NEE)
